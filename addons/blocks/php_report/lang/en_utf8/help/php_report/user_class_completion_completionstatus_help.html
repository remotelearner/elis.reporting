User Class Completion Report Help

This field only allows classes which match the selected value in the report.

There are four values for this field:

  1. Passed: The user has passed the class.
  2. Failed: The user has failed the class.
  3. In Progress: The user has logged into the class or completed one or more course elements*.
  4. Not Started: The user has never logged into the class and has not completed any course elements*.

* Course elements may be marked as completed by course instructors.
